# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive integration documentation for Prisma AIRS (AI Runtime Security)
- Step-by-step setup guides for third-party platform integrations
- Integration guides for 7 platforms: Anthropic, Kong, LiteLLM, n8n, Portkey, TrueFoundry
- Multiple Kong integration options (custom plugin and request callout)
- Detailed prerequisites and configuration steps for each integration
- External resource links to Prisma AIRS documentation

### Integration Platforms
- **Anthropic**: Foundation model provider integration
- **Kong**: API gateway integration with custom plugin and request callout options
- **LiteLLM**: LLM gateway integration
- **n8n**: Workflow automation platform integration
- **Portkey**: AI gateway and observability integration
- **TrueFoundry**: AI gateway integration

### Documentation
- Central repository for all Prisma AIRS integrations
- Key concepts documentation (AIRS, Strata Cloud Manager, Security Profiles, Guardrails)
- Links to Prisma AIRS Developer Documentation and Administrator Guide
- Contributing guidelines for adding new integrations

### Features
- Inline, real-time security for AI applications
- Protection against prompt injection, malicious code, and data leakage
- Security profile configuration through Strata Cloud Manager
- Guardrail implementation across multiple platforms
- Enterprise-grade AI security integration framework
